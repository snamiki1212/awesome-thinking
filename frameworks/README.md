# 思考フレームワーク 一覧

決まった項目・軸に沿って情報を当てはめ、整理・分析するための「枠組み・テンプレート」。
タイプの定義は [思考の分類と定義](../docs/taxonomy.md) を参照してください。

| 項目 | 別名 | ひとこと |
| --- | --- | --- |
| [SWOT分析](./swot.md) | SWOT | 強み・弱み・機会・脅威の4象限で現状を把握 |
| [MECE](./mece.md) | モレなくダブりなく | 漏れなく重複なく分ける分類の原則 |
| [ロジックツリー](./logic-tree.md) | Logic Tree | テーマを木構造で階層的に分解する |
| [5W1H](./5w1h.md) | — | 6つの問いで情報の漏れを防ぐ |
| [PDCAサイクル](./pdca.md) | Plan-Do-Check-Act | 計画→実行→評価→改善を回し続ける |
| [KPT](./kpt.md) | Keep-Problem-Try | 続ける・課題・次に試すで振り返る |
| [なぜなぜ分析](./5-whys.md) | 5 Whys | 「なぜ」を繰り返して根本原因に迫る |
