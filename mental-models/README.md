# メンタルモデル 一覧

世界や物事の仕組みを理解・予測するための「レンズ」。それ自体は手順を持たず、状況の見方・前提として働きます。
タイプの定義は [思考の分類と定義](../docs/taxonomy.md) を参照してください。

| 項目 | 別名 | ひとこと |
| --- | --- | --- |
| [第一原理思考](./first-principles.md) | First Principles | 前提を疑い、根本の事実まで遡って考え直す |
| [機会費用](./opportunity-cost.md) | Opportunity Cost | 選ぶことは、選ばなかった価値を捨てること |
| [複利](./compound-interest.md) | Compound Interest | 成果が次の成果の土台になり、伸びが加速する |
| [地図は領土ではない](./map-is-not-the-territory.md) | Map ≠ Territory | モデルやデータは現実そのものではない |
| [パレートの法則](./pareto-principle.md) | 80:20の法則 | 成果の大部分はごく一部の要因から生まれる |
| [センターピン](./center-pin.md) | センターピン理論 / Center Pin | 倒せば残りも連鎖して倒れる急所の一点を狙う |
