# 2グループの絵文字を確定する（レンズ=🔍 / プロセス=⚙️）

## ステータス
Accepted（採用）

## 背景・課題
[20260604-lens-vs-process-grouping](../20260604-lens-vs-process-grouping/README.md) で、
思考ツール（4タイプ）の最上位の切れ目を **「🔍 レンズ（1）／⚙️ プロセス（3）」** の2グループに
大別することを決めた。

4タイプには確定済みの判別絵文字（🧠 / 🧩 / 🌊 / ⚡。
[20260604-type-emoji](../20260604-type-emoji/README.md)）があるが、
**2グループ（レンズ・プロセス）の側には絵文字がなく**、図・見出しでグループを視覚的に示せなかった。
グループ名にも一目で伝わるアイコンを与えたい。

## 決定
2グループの絵文字を次のとおり確定する。

| グループ | 絵文字 | 含むタイプ | 意味 |
| --- | --- | --- | --- |
| レンズ | 🔍 | 🧠 メンタルモデル | 見る・捉える（静的な視点） |
| プロセス | ⚙️ | 🌊 思考法 ⊃ 🧩 フレームワーク ⊃ ⚡ 思考術 | 進める・動かす（抽象→具体） |

一覧・見出し・図ではこの絵文字で統一する。正本は [`CLAUDE.md` 用語節](../../CLAUDE.md)。

## 理由
- **🔍（虫眼鏡）＝「見る・レンズ」** が、レンズの「状況を見る・捉える視点」をそのまま表す。グループ名「レンズ」とも一致。
- **⚙️（歯車）＝「仕組み・プロセスが動く」** が、プロセスの「考えを進める・手を動かす」を端的に表す。グループ名「プロセス」と相性がよい。
- 4タイプの絵文字（🧠 / 🧩 / 🌊 / ⚡）と**形・モチーフが重ならず**、グループの絵文字とタイプの絵文字を取り違えにくい。
- 専門知識がなくても意味が伝わる平易なアイコンで、リポジトリのトーンに合う。

## 検討した代替案

### レンズ（メンタルモデル＝見る・捉える）
| 候補 | 評価 |
| --- | --- |
| 🔍 虫眼鏡 | **採用**。「見る・レンズ」を最も直感的に表し、グループ名とも一致 |
| 👁️ 目 | 見送り。意味は近いが、単体の目はやや生々しく汎用ラベル向きでない |
| 🔭 望遠鏡 | 見送り。「遠くを見通す・観察」だが「レンズ」より範囲が限定的に読まれる |

### プロセス（思考法・フレームワーク・思考術＝進める・動かす）
| 候補 | 評価 |
| --- | --- |
| ⚙️ 歯車 | **採用**。「プロセス・仕組みが動く」を端的に表す |
| 🔄 循環矢印 | 見送り。「進める・回す」は表せるが、PDCA など循環系フレームワークと意味が被りやすい |
| 🧰 道具箱 | 見送り。「道具一式」だが総称「思考ツール（道具）」と紛らわしい |

## 影響
- README・`CLAUDE.md` の用語・図で、グループ名に 🔍 / ⚙️ を併記して統一する。
- 4タイプの絵文字（[20260604-type-emoji](../20260604-type-emoji/README.md)）とは別レイヤーの「グループラベル」として用いる。
- ディレクトリ構成・タイプ定義は変更しない（視覚ラベルの取り決めのみ）。
