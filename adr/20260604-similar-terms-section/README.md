# 詳細ファイルに「似ている用語との違い」節を追加する

## ステータス
Accepted（採用）

## 背景・課題
思考概念には、名前や効果が似ていて混同されやすいものが多い。
例えば「複利」と「レバレッジ」は、どちらも「効果を増幅する」点で似ているが、
複利は時間をかけて積み上がるもの、レバレッジは他者の資源で一度に拡大するもの、という決定的な違いがある。

既存テンプレートには `関連` 節しかなく、ここは「一緒に使える・つながる項目」を相互リンクする場であって、
「似ているが別物の概念を切り分ける」用途には合っていなかった。結果として、読者が混同しやすい点を解消する場所がなかった。

## 決定
詳細ファイルのテンプレート（`add-thinking-concept` スキルが正本）に、
`注意点・落とし穴` と `関連` の間へ **`## 似ている用語との違い`** 節を新設する。

- 名前や効果が似ていて混同されやすい概念を挙げ、「どこが似ていて・どこが違うか」を一言で示す。
- リポジトリに未収録の用語を挙げてもよい。該当項目があれば `[[スラッグ]]` でリンクする。
- 思い当たらなければ「特になし」と明記する（空欄にしない）。

## 理由
- 混同の解消は、思考概念を一覧管理するうえで読者価値が高い。
- `関連`（つながる項目の相互リンク）と `似ている用語との違い`（紛らわしい別概念の切り分け）は目的が異なるため、節を分けたほうが書き手も読み手も迷わない。

## 検討した代替案
- **`関連` 節に混ぜて書く**: つながりと差異が混在して読みにくく、相互リンクの運用ルールも崩れるため却下。
- **`注意点・落とし穴` に含める**: あくまで当該概念単体の誤用注意であり、他概念との対比とは粒度が異なるため却下。

## 影響
- 今後追加する項目は新節を埋める（似た用語が無ければ「特になし」）。
- 既存項目への遡及適用は必須としない。気づいたものから順次補う。
- スキル `add-thinking-concept` の SKILL.md（テンプレート・執筆スタイル・完了チェックリスト）を更新済み。
