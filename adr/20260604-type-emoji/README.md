# 4タイプの判別絵文字を確定する（思考法=🌊 / 思考術=⚡）

## ステータス
Accepted（採用）

## 背景・課題
一覧表・図・見出しで、4タイプを絵文字で視覚的に判別している。しかし従来の割り当て——

- 🧠 メンタルモデル
- 🧩 思考フレームワーク
- 🧭 思考法
- 🛠 思考術

のうち、**🧭（思考法）と 🛠（思考術）がどちらもグレー系**で、表に並ぶと一目での区別がつきにくかった。絵文字をジャンルの「色ラベル」として機能させるには、4つが**互いに異なるはっきりした色**で並ぶ必要がある。

## 決定
4タイプを、互いに色が十分離れた絵文字で確定する。

| タイプ | 絵文字 | 主な色 |
| --- | --- | --- |
| メンタルモデル | 🧠 | ピンク |
| 思考フレームワーク | 🧩 | 緑（環境により多色） |
| 思考法 | 🌊 | 青 |
| 思考術 | ⚡ | 黄 |

この4つを正式な割り当てとし、一覧・見出し・図・スキルの記載で統一する。正本は [`docs/taxonomy.md`](../../docs/taxonomy.md) と [`CLAUDE.md` 用語節](../../CLAUDE.md)。

## 理由
- **色の分離が今回解くべき課題**だった。🧠（ピンク）・🧩（緑）・🌊（青）・⚡（黄）は4色が四方に離れ、表でも瞬時に見分けられる。
- 絵文字はここでは**常に「思考法」「思考術」というテキストラベルと並んで**表示される。意味の正確さはラベルが担保するため、絵文字側は「色での識別」を優先してよい。
- 意味の整合も最低限とれている：🌊 は「潮流・流れ＝どの方向から攻めるか（流儀）」=思考法に、⚡ は「即座に打てる一手・瞬発」=思考術（その場の一手）に対応づけられる。

## 検討した代替案

### 思考法（方向・姿勢・流儀／抽象）
当初は 🎯（赤）を選んだが、**🧠（ピンク）と同じ暖色系で近く見える**という問題が判明し、寒色系で選び直した。

| 候補 | 評価 |
| --- | --- |
| 🌊 波（青） | **採用**。寒色で 🧠・⚡ と明確に分離。「潮流・流れ」が流儀・方向に通じる |
| 🎯 ターゲット（赤） | **却下**。意味は良いが赤主体で 🧠（ピンク）と色が近く、判別目的に反する |
| 🔮 水晶玉（紫） | 寒色で分離は良いが「占い・予測」と読まれやすい |
| 🔭 望遠鏡 | 「どの方向・視点から見るか」と意味は最良だが、色が暗く色分け効果が弱い |
| 🧭 コンパス（灰） | 却下（従来案）。意味は最良だが灰色で 🛠 と見分けにくい |

### 思考術（その場の一手／具体）
| 候補 | 評価 |
| --- | --- |
| ⚡ 稲妻（黄） | **採用**。黄で目立ち、「即座の一手」を表せる |
| ✨ きらめき（黄） | 「ひらめき」寄りで、発想系の思考法と意味が被りやすい |
| 🔑 鍵（金） | 「詰まりを開ける一手」だが色味が弱く判別性で劣る |
| 🛠 工具（灰） | 却下（従来案）。意味は最良だが灰色で 🧭 と見分けにくい |

## ゼロベースでの再検討（本当にこれでいいか）
選定後に「そもそもこの割り当てで妥当か」を白紙から問い直し、2点の見直しを行った。

- **意味 vs 色のトレードオフ**: *意味的に最も適切な絵文字（🧭 コンパス＝方向／流儀、🛠 工具＝技／操作）が、まさに今回はじいたグレーの2つ*だった。本件は「意味の正確さ」と「色の見分けやすさ」のトレードオフであり、今回は解くべき課題に直結する後者を優先した。
- **暖色の近接（再検討で修正）**: 思考法に一度 🎯（赤）を当てたが、🧠（ピンク）と暖色系で近く、小さい表示では混同しうる。さらに 🧩 は環境により緑で表示されるため、思考法は**寒色（青）= 🌊** とし、ピンク／緑／青／黄で四隅に色を散らした。
- **🌊 の懸念**: 「海・水」と字義どおり読まれ、流儀の比喩が伝わりにくい余地がある。→ 常に「思考法」ラベルと併記され、絵文字は色コードとして働くため実害は小さいと判断。
- **結論**: 🧠 / 🧩 / 🌊 / ⚡ を採用。理由は (1) 解くべき課題が「色の分離」であること、(2) 絵文字は常にテキストラベルと並ぶため意味は二次的でよいこと、(3) 寒色を1つ入れることで4色が最も離れること。
- **再検討の条件（将来の見直しトリガー）**: 絵文字単独（ラベルなし）で表示する UI を導入する場合は、意味の取り違えリスクが顕在化するため、本決定を見直す。

## 影響
- README / `docs/taxonomy.md` / スキル `add-thinking-concept` の記載を 🌊・⚡ に統一した。
- 既存 ADR `20260604-single-table-index` の例示も新しい絵文字に合わせて表記を更新した（決定内容は不変）。
- 今後の項目追加では、一覧行のジャンルラベルにこの4絵文字を用いる。
