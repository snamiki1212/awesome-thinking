# 思考術 一覧

日常・実務で使える具体的な「技・コツ」。粒度が小さく、その場ですぐ実践できる操作レベルのテクニックです。
タイプの定義は [思考の分類と定義](../docs/taxonomy.md) を参照してください。

| 項目 | 別名 | ひとこと |
| --- | --- | --- |
| [抽象化と具体化](./abstraction-and-concretization.md) | 抽象と具体の往復 | 本質を抜き出し、別の場面に当てはめ直す |
| [ゼロベース思考](./zero-based-thinking.md) | 白紙思考 | 前提を白紙に戻し「今ゼロから始めるなら」と考える |
| [悪魔の代弁者](./devils-advocate.md) | Devil's Advocate | あえて反対役になり、弱点をあぶり出す |
| [リフレーミング](./reframing.md) | 枠組みの捉え直し | 同じ事実を別の枠組みから捉え直す |
| [極端思考](./extreme-case-thinking.md) | 極端な例で考える | 変数を両極端まで振り切り、本質や効きどころを浮かび上がらせる |
| [命名](./naming.md) | ネーミング | 名前のない事象に呼び名を与え、思考と会話で扱えるようにする |
| [思考実験](./thought-experiment.md) | Thought Experiment | 頭の中に仮想状況を設定し、帰結を推論して前提を吟味する |
| [虫の目・鳥の目・魚の目](./birds-worms-fish-eye.md) | 三つの目 | 細部・全体・流れの3視点に切り替えて多面的に捉える |
| [アクティブリコール](./active-recall.md) | ブラーティング / 検索練習 | 見ずに思い出して書き出し、理解の穴を見つける |
