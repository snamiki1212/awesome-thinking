# 思考法 一覧

物事に向き合うときの「考え方の流儀・スタイル」。思考全体の方向性やモードを決める上位の構えです。
タイプの定義は [思考の分類と定義](../docs/taxonomy.md) を参照してください。

| 項目 | 別名 | ひとこと |
| --- | --- | --- |
| [ロジカルシンキング](./logical-thinking.md) | 論理的思考 | 主張と根拠を筋道立てて結びつける |
| [クリティカルシンキング](./critical-thinking.md) | 批判的思考 | 前提や根拠を吟味し、思い込みを排する |
| [ラテラルシンキング](./lateral-thinking.md) | 水平思考 | 前提を飛び越えて新しい発想を生む |
| [システム思考](./systems-thinking.md) | Systems Thinking | 要素のつながり・全体構造で捉える |
| [デザイン思考](./design-thinking.md) | Design Thinking | 共感を起点に試作と検証で解を磨く |
| [仮説思考](./hypothesis-thinking.md) | Hypothesis Thinking | 仮の答えを立て、検証しながら進める |
| [ネガティブ・ケイパビリティ](./negative-capability.md) | Negative Capability | 相反する見通し・不確実さを未解決のまま抱え続ける |
| [エッセンシャル思考](./essentialism.md) | Essentialism | 本当に重要な少数を見極め、それ以外を削ぎ落として集中する |
